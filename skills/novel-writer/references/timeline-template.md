# 时间线模板

> 记录故事内的时间流逝和关键事件。用于一致性巡检。

## 时间线

| 故事时间 | 章节 | 事件 | 备注 |
|---------|------|------|------|
| 第1天 上午 | 第1章 | 主角觉醒异能 | 故事起点 |
| 第1天 下午 | 第1章 | 首次遭遇敌人 | |
| 第2天 | 第2章 | 加入组织 | |
| 第3天-第7天 | 第3-4章 | 训练期 | 时间跳跃 |
| 第8天 | 第5章 | 首次任务 | |
