# 世界观构建模板

## 使用说明

此模板供世界观构建时参考。实际项目中，世界观按维度拆分为多个文件存储在 `novel/[项目名]/worldbuilding/` 目录下，文件名按题材自由命名。不必照搬所有板块——只创建你的题材需要的部分。
