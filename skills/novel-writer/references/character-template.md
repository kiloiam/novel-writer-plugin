# 角色档案格式

每个角色一个文件：`characters/[角色名].md`

```markdown
# [角色名]

## 基本信息
- **身份**：
- **年龄**：
- **外貌**：一句话速写（用于正文描写时保持一致）

## 性格
- **关键词**：（3-5个，如：隐忍、腹黑、护短）
- **说话风格**：（如：话少，句句有分量；爱反问）
- **底线**：（绝对不做的事）

## 能力
- **当前等级**：
- **核心技能**：
- **持有道具**：

## 关系网
- A → 关系描述
- B → 关系描述

## 成长记录
> 只记录有实质变化的章节，不是每章都记

### 第X章
- 变化内容（等级、关系、心态、获得道具等）

### 第Y章
- 变化内容
```
